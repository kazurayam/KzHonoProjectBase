# Hono-API-Server


[TypeScript初心者の私がHonoでバックエンドサーバー構築してみた](https://zenn.dev/yuzunosk55/articles/09275c72cf051b) を写経する

## 操作

### サーバーを起動する

```
$ cd Hono-API-Server/my-app
$ bun run dev
```

### クライアントを起動する

サーバーとは別のTerminalで

```
$ cd Hono-API-Server/my-app
$ bun run client
```

clientプロセスがholdする。それを停止するには Ctrl+C する。
